This web page is a location for finding accessible parking in Seattle. The user can filter out results using the controls in the sidebar. By default, all parking garages in the City of Seattle are displayed on the map; however, the user can toggle more parking types. The results on the map can be further filtered by providing a location around which to search for traffic within a given radius. These results can be further filtered by time contraints by only showing parking locations that are open/available at the given time. If no time or date is provided, parking locations that are currently open will be shown.
