App Usage

This web page is a location for finding accessible parking in Seattle. There are 4 different types of parking data that can be viewed. These include parking lots, parking garages and e-parking lots; parking pay stations which correspond to paid street parking locations; and restricted parking zones (RPZs). The 4th option is to view the street parking options that exist in the city. These options essentially provide a low level view of the different parking curbspace categories for the City of Seattle and can take some time to load and render on the map. YOU HAVE BEEN WARNED...

Using the controls in the sidebar, you can view different parking types when their data has been loaded onto the page. The sidebar also allows for making searching for parking near a given location and near you easy by allowing you the ability to define a location and a radius around which to highlight parking. You also have the ability to view the payment status of pay stations based on a given time and day.

Data

The data on this web page is provided by the City of Seattle through their ArcGIS Server map services and is provided as is with no guarantee of accuracy.
